# Introduction

Pyth Network is an oracle protocol that offers several products to developers:

- [Price Feeds](../price-feeds) provide real-time prices for 40+ blockchain ecosystems, including Solana, many EVM chains,
  Aptos, Sui, and several Cosmos chains.
- [Benchmarks](../benchmarks) provides historical Pyth prices for both on- and off-chain use.
- [Entropy](../entropy) allows developers to generate secure random numbers on the blockchain.

Developers who are interested in using these products can follow the links above to access product-specific documentation.

Other useful links may be:

- [Pyth Metrics](home/metrics) -- See metrics of the network's adoption and growth.
- [Pyth Token](home/pyth-token) -- Learn more about the Pyth governance token PYTH.
